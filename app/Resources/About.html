What is MIDI Boy

MIDI boy can be useful for:

  • Controlling MIDI device such as a synthesizers without the requirement of a heavy DAW
  • Editing hardware patch without using a device dedicated virtual interface
  • Reading and writing dumps from hardware using MIDI
  • Monitoring MIDI messages

MIDI boy features

  • Monitor MIDI input ports

    • Note On
    • Note Off
    • Control Change
    • Program Change
