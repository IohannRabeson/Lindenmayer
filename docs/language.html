Programming in L-Code

Structure of a L-Code program

The L-Code language is an attempt to provide a language able to describe several Lindenmayer's systems.

An L-Code program look like almost any Lindenmayer's system textual representation. The following example is the Koch curve written in L-Code: {% highlight text %} iteration: 4; distance: 10.; angle: 90.; axiom: --F; F -> F+F-F-F+F; {% endhighlight %} This code is very similar to the Koch curve pseudo-code on Wikipedia: {% highlight text %} variables : F constants : + − start : F rules : (F → F+F−F−F+F) {% endhighlight %} And finally the image generated by Lindenmayer:
